网站简介

思政大课堂是由HTML和CSS写出来的一个静态网页，和用户之间不能进行交互。

思政大课堂网页由五部分组成。分别是导航栏，版心，学习推荐，精品模块，footer。其中版心的所有链接均链接到了其它的学习网站。 其余部分的链接未做任何链接处理。

由于本人水平有限，网页中当有许多不合适甚至错误的地方，如果误人子弟那就太不美了。在之后，如果有时间和精力，我还会继续学习，补充完善思政大课堂的内容，加入JS技术，实现与用户的交互。同时我将它放在Github 上希望更多的人来完善补充，提出有益的建议。
